# ⚛️ Footer

Provides a standard website or app footer.

## Types

| Type    | Usage   |
| ------- | ------- |
| MAXIMAL | Maximal |
| MINIMAL | Minimal |

## Usage

```js
<script>
  import Footer from "@dusk-network/footer";
  import { types } from "@dusk-network/helpers";

  },
  {
    title: "LinkedIn",
    icon: "linked-in",
    url: "https://www.linkedin.com/company/dusknetwork/",
  },
  {
    title: "Github",
    icon: "github",
    url: "https://github.com/dusk-network",
  },
];

const categories = [
  {

    links: [
```
